Install Dumbify

Requires Chrome or Edge on desktop. Phone must be running Android 10+.

Unsupported browser

WebUSB is required and only works in Chrome or Edge on desktop (Windows, macOS, Linux, ChromeOS).

Firefox, Safari, and all mobile browsers do not support WebUSB.

Enable USB debugging

Follow these steps on your Android phone before connecting via USB.

  1. Open Settings → About phone
  2. Tap Build number 7 times until you see "You are now a developer!"
  3. Go back to Settings → Developer options
  4. Enable USB debugging
  5. Connect phone to this computer with a USB cable

Menu names vary by manufacturer. See device-specific instructions.

Connect & authorize

Click the button below. A browser dialog will ask you to pick your phone. After selecting it, tap "Allow USB debugging" on the phone.

Pre-flight checks

Verifying Android version and that no Google accounts are signed in (required for Device Owner).

Google accounts detected

Android requires no accounts before a Device Owner can be set.
Go to Settings → Accounts on the phone and remove all accounts, then click Retry.

Download & push APK

Fetching the latest Dumbify release from GitHub and pushing it to your device.

Install APK

Running pm install on the device.

Set Device Owner

This gives Dumbify the power to hard-block apps at the OS level.

Another Device Owner is already set

Android only allows one Device Owner. To proceed you must factory-reset the phone, which removes all data. See FAQ: DO conflict.

🎉

All done!

Disconnect the USB cable.
Open the Dumbify app on your phone and run the setup wizard.
