How it works

Dumbify runs as the Android Device Owner — the same privilege level as MDM software used in enterprises. This gives it the power to suspend apps at the OS level, not just hide them.

  1. Connect your phone via USB to a desktop browser (Chrome/Edge)
  2. The installer site pushes the APK and sets Device Owner via WebUSB ADB
  3. Run the setup wizard — pick blocked apps, set PINs, configure bypass friction
  4. Hand the removal PIN to a trusted person
